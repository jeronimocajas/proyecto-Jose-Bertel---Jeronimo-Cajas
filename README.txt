# Curso de Ingenieria Economica

Este proyecto contiene una página web educativa sobre Ingenieria Economica, cubriendo temas como series uniformes, series variables y evaluación de inversiones.

## Estructura del Proyecto

- **/assets**: Recursos del proyecto (CSS, JS, imágenes, descargas).
  - **/css**: Hojas de estilo.
  - **/js**: Scripts de JavaScript.
  - **/images**: Imágenes.
  - **/downloads**: Archivos descargables, como plantillas de Excel.
- **/pages**: Páginas HTML del sitio web.
  - `index.html`: Página de inicio.
  - `teoria.html`: Página de teoría.
  - `ejercicios.html`: Página de ejercicios.
  - `herramientas.html`: Página de herramientas.
  - `normatividad.html`: Página de normatividad.
  - `contacto.html`: Página de contacto.

## Cómo Usar

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador para comenzar a navegar por el sitio web.
